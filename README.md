# Reality Design System (RDS) - by Gong.io

Welcome to the Reality Design System (RDS), a foundational design system for creating consistent, high-quality landing pages and marketing materials for Gong.io.

## Core Principles

1.  **Clarity First:** The design must be intuitive and easy to understand.
2.  **Confident & Data-Driven:** The visual language should feel authoritative and trustworthy.
3.  **Efficient & Scalable:** Components are reusable, enabling teams to build quickly.

## How to Use

1.  **Link the CSS:** Add the stylesheet to the `<head>` of your HTML file.
    ```html
    <link rel="stylesheet" href="css/reality.css">
    ```

2.  **Add the Font:** RDS uses the 'Lexend' font from Google Fonts. Add this to your `<head>` as well.
    ```html
    <link rel="preconnect" href="[https://fonts.googleapis.com](https://fonts.googleapis.com)">
    <link rel="preconnect" href="[https://fonts.gstatic.com](https://fonts.gstatic.com)" crossorigin>
    <link href="[https://fonts.googleapis.com/css2?family=Lexend:wght@400;500;600;700&display=swap](https://fonts.googleapis.com/css2?family=Lexend:wght@400;500;600;700&display=swap)" rel="stylesheet">
    ```
3.  **Use the Components:** Use the provided CSS classes to build your layout. See `index.html` for a complete example.

## Core Components

-   `.btn`, `.btn-primary`, `.btn-secondary`
-   `.card`
-   `.form-group`, `.form-control`
-   `.tag`

Utility classes for spacing (e.g., `.mt-4`, `.py-8`) and layout (`.container`, `.d-flex`) are also available.

---
*This is a foundational system. It is intended to be expanded with more components, patterns, and documentation over time.*
